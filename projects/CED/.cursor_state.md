# Состояние проекта

## Фаза: 4 — Единый Ubuntu-сервер (без Windows-шлюза)
## Обновлено: 2026-06-01

## Архитектура (утверждена)
- **Сервер:** Ubuntu, Docker Compose (`docker-compose.yml` + `docker-compose.server.yml`)
- **ИИ:** Ollama на том же хосте (мост `ollama-docker-bridge.sh` для контейнеров)
- **Каталог КД:** SMB/CIFS mount после OpenVPN → `CED_CATALOG_MOUNT`
- **Клиенты:** Windows `KdCatalog.exe` (:8000), браузер (:80, `/api`)
- **Не используется:** Windows portable-шлюз, SSH-туннель к Ollama

## Стек
- Backend: Python 3.11, FastAPI, Celery, PostgreSQL 15, Redis 7
- AI: ai-agent + Ollama
- Web: Vue 3 (production build → `/api` через nginx)
- Desktop: .NET 8 WinForms client only
