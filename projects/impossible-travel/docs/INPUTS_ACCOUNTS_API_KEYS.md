# API Keys & Accounts

Скопируйте `.env.example` в `.env` и заполните ключи по мере подключения v2.

| Сервис | Переменная | MVP |
|--------|------------|-----|
| OpenAI | OPENAI_API_KEY | не требуется |
| Anthropic | ANTHROPIC_API_KEY | не требуется |
| Gemini | GEMINI_API_KEY | не требуется |
| Runway | RUNWAY_API_KEY | не требуется |
| ElevenLabs | ELEVENLABS_API_KEY | не требуется |

При `MOCK_GENERATION=true` все генерации идут через mock-провайдеры.
