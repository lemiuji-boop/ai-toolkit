# Roadmap

## MVP (текущая версия)

- Mock AI providers
- Полный episode pipeline через Celery
- Web dashboard: create → generate → review → approve
- Compliance report + AI disclosure

## v2

- Реальные text/image/video/TTS API
- FFmpeg монтаж, субтитры, MP4 export
- Календарь публикаций, Telegram

## v3

- Reference images для стабильного лица
- A/B тесты, аналитика удержания
- Cost control, batch generation

## v4

- Сезонные арки, платные маршруты
- Сайт, рекламный кабинет, медиакит
