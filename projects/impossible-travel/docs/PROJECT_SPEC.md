# Project Spec — Impossible Travel AI Blog

Виртуальный тревел-блог о местах, куда невозможно попасть. Главная героиня — **Мира** (виртуальный блогер, не копия реального человека).

## Формат

- Вертикальное видео 9:16, 30–90 сек
- Язык: RU (+ EN для packaging)
- Стиль: photorealistic cinematic travel vlog

## MVP Pipeline

```text
Идея → локация → мир → персонажи → сценарий → раскадровка → промты
→ mock assets → social package → compliance → review → approve
```
