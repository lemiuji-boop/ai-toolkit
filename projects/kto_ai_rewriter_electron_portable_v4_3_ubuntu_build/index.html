Вставка вопросов

Можно вставить вопросы построчно. Для автора используйте формат: вопрос → Tab → Фамилия И.О.

Удобный порядок работы:

1. Вставьте вопросы из Excel.
2. Укажите автора, если он не подставился.
3. Выберите провайдера.
4. Нажмите «Обработать выбранные».
5. Проверьте результат и экспортируйте CSV.

Настройки API‑ключей закрыты

Вы работаете как гость. Ключи провайдеров, их просмотр и изменение доступны только администратору по паролю.

Администрирование провайдеров и API‑ключей

Ключи и основной промпт видны только после входа администратора. На сервере они сохраняются в файле server-data.json. Для промышленной эксплуатации храните файл на защищенном сервере, ограничьте доступ к папке проекта и задайте сложный ADMIN_PASSWORD.
Основной промпт редактора

Этот промпт применяется ко всем пользователям при AI‑редактировании вопросов. Гости его не видят и не меняют.

Дополнительные правила из левой панели добавляются к этому промпту в конце. Если нужно полностью изменить стиль редакции, меняйте именно основной промпт.
