# Надстройка МАТНОРМ-XL (фронтенд для Vercel)

Статическая Office-надстройка (Office.js). Хостится на Vercel; обращается к
on-prem бэкенду по адресу, который пользователь вводит в панели.

## Деплой на Vercel
```bash
npm i -g vercel
cd addin
vercel        # привязать проект; задеплоит статику
vercel --prod # продакшн-URL вида https://your-addin.vercel.app
```
После деплоя подставьте URL в `manifest.xml` (SourceLocation) и в CORS бэкенда.

## Раздача пользователям без прав администратора
Office-надстройка НЕ устанавливается как программа — поэтому права админа на ПК
пользователя не нужны. Варианты раздачи:
1. **Централизованное развёртывание (рекомендуется).** Админ M365 (Integrated apps /
   Centralized Deployment) загружает manifest.xml и назначает пользователям —
   надстройка появляется в Excel автоматически.
2. **Сетевой каталог надстроек.** Положить manifest.xml в общую папку; пользователь
   один раз добавляет путь в Excel → Параметры → Центр управления безопасностью →
   Каталоги надстроек (это пользовательская настройка, админ не требуется).
