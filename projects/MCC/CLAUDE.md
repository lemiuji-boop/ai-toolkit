# Правила для AI-агента — проект МАТНОРМ

Источник правды — `docs/TZ.md`. Реализовывать только FR из ТЗ; спорное — в раздел 14
ТЗ «Открытые вопросы», не в код.

1. Каждый FR прослеживается до кода и теста (`docs/trace.md`).
2. LLM не выполняет нормативную арифметику — нормы/КИМ/коэффициенты/входимость
   считает только детерминированный код (`app/services/calc.py` + `app/data/rules.json`).
3. On-premise: запрет исходящих вызовов внешних API. Секреты — через переменные окружения.
4. ОС — Ubuntu, скрипты bash-совместимы.
5. Стек и версии зафиксированы в `pyproject.toml`.
6. Перед коммитом: `ruff check . && mypy app && pytest -q` зелёные.
7. Один FR за итерацию: реализация → тест (без GPU/LLM) → строка в `docs/trace.md`.
8. Комментарии в коде — на русском языке; type hints и Pydantic-схемы обязательны.

Подробнее — `.cursor/rules/*.mdc`.
