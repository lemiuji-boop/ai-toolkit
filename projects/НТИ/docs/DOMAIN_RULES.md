# Правила домена НТИ.Сбор

## Перевод единиц в н/ч (FR-011)

| Единица | Формула |
|---------|---------|
| н/ч | значение как есть |
| мин | значение / 60 |
| чел.-ч | значение как есть (эквивалент н/ч для сводки MVP) |

## Форматы (NFR-007)

- Дата отображения: `ДД.ММ.ГГГГ`
- Десятичный разделитель ввода: запятая
