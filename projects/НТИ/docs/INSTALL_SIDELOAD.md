# Установка НТИ.Сбор (sideload)

## Сборка release APK

```bash
cd android
./gradlew :app:assembleProdRelease
```

APK: `app/build/outputs/apk/prod/release/app-prod-release.apk`

Подпись: настройте `signingConfigs` в `app/build.gradle.kts` с keystore предприятия (не храните keystore в Git).

## Установка на устройство

1. Android 8.0+ (API 26).
2. Настройки → Безопасность → разрешить установку из неизвестных источников (для файлового менеджера или ADB).
3. Скопируйте APK на устройство и откройте файл, либо:

```bash
adb install -r app-prod-release.apk
```

## Первый запуск

1. Настройки → укажите ФИО исполнителя (FR-009).
2. При необходимости — адрес сервера НТИ и токен.
3. Главный экран → «+» для новой записи.

## Матрица проверки (раздел 5 ТЗ)

- [ ] Экран ~5" (360 dp)
- [ ] Крупный телефон
- [ ] Планшет
- [ ] Шрифт 100% / 200%
- [ ] Светлая / тёмная тема
- [ ] Портрет / ландшафт
