# MASTER PROMPT FOR CURSOR

Ты работаешь над проектом AI-МАТНОРМ. Нужно разработать весь стек с нуля, строго по документам в папке `docs`.

Главная цель: создать промышленный ассистент технолога для анализа КД, построения КСИ и расчёта норм материалов.

## Обязательные правила

1. Не используй старый MVP как основу архитектуры.
2. Разрабатывай поэтапно.
3. После каждого этапа проект должен запускаться.
4. Backend, frontend, desktop, infra, docs должны быть разделены.
5. Все долгие операции выполняй через очередь задач.
6. Все LLM-ответы валидируй через JSON Schema/Pydantic.
7. Все расчёты материалов выполняй детерминированно, не через LLM.
8. Все действия пользователя логируй.
9. Все расчёты версионируй.
10. Не храни секреты в frontend.

## Первый этап

Начни с `prompts/SPRINT_01_BOOTSTRAP.md`.

Перед кодированием:

- прочитай `docs/AI_MATNORM_GREENFIELD_TZ.md`;
- прочитай `docs/IMPLEMENTATION_PLAN.md`;
- предложи структуру файлов;
- затем реализуй Sprint 01.
